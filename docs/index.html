    1. Ungrid

    For Flexible Grid We have used ungrid in our layout, you can find more about it Here

    You can create row using .table-row class.

    To insert column inside row you can use .cell class.

    3. Navbar

    By default navigation bar have relative positioning and transparent background.you can easily modify it's behaviour by appending extra classes.

    2. Timer Code

    Timer code will generate inside this html tag.you can change countdown to specific date by set it inside data-countdown attribute.

    <span class="timer" data-countdown="2017/08/13"></span>

    JS Code that powers current timer is below.

    $('[data-countdown]').each(function() {
    var $this = $(this),
        finalDate = $(this).data('countdown');
    $this.countdown(finalDate, function(event) {
        $this.html(event.strftime('' + 
        '<span data-type="days">%D</span> ' + 
        '<span data-type="hours">%H</span>:' + 
        '<span data-type="minutes">%M</span>:' + 
        '<span data-type="seconds">%S</span>'));
    });
});

    in above code data-type define text to be inserted below timer, it can be anything. For Timing parameters you can use %D, %H, %M, %S and many others.

    You can find documentation of all parameters here
